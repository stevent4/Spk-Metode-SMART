# Spk-Metode-SMART

SPK SMART sapi berkualitas adalah sebuah sistem pendukung keputusan (SPK) yang digunakan untuk menentukan kualitas sapi yang akan diproduksi oleh peternak. SPK ini mengadopsi metode SMART (Simple Multi-Attribute Rating Technique) dalam mengevaluasi kriteria-kriteria yang berkaitan dengan kualitas sapi.

# Kriteria Evaluasi
Berikut adalah kriteria-kriteria yang digunakan dalam SPK ini:

Kaki
Telinga
Punggung
Ekor
Punuk
Bibir
Pantat

# Cara Menggunakan
Untuk menggunakan SPK SMART sapi berkualitas, ikuti langkah-langkah berikut:

Clone repository ini ke komputer lokal Anda.
import file sql ke dalam localhost/phpmyadmin anda.
Masukkan data sapi yang ingin Anda evaluasi sesuai dengan kriteria yang telah ditentukan.
